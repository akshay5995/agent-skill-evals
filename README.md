# SkillKit

Promptfoo-native plugin kit for evaluating reusable agent skills.

See [SPEC.md](./SPEC.md) for the full specification.

## Status

v0.1 — under active development. Phases ship sequentially per the
implementation plan; each phase is verified by `npm test` and a
Promptfoo end-to-end run before commit.

## Quick start (for contributors)

```bash
npm install
npm run typecheck
npm test
npm run build
```
